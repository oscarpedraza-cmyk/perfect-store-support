# Perfect Store Support

Herramienta web independiente y **no oficial de Rappi** que ayuda a aliados de restaurantes (de cualquier país) a subir fotos de producto, portada, logo y toppings sin errores.

Analiza formato, peso, dimensiones, proporción, brillo, saturación, nitidez y encuadre (con un modelo de IA de recorte de fondo que corre 100% en el navegador — ninguna imagen se sube a un servidor), y puede corregir automáticamente la foto (recorte, resize, compresión, ajuste de brillo/saturación, fondo blanco para toppings).

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
